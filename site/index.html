Overview

A classic puzzle loop, adapted for desktop play

What it is

Chris' Puzzle Game is a classic sliding tile game: one square is empty, the rest are scrambled, and you slide adjacent pieces until the full image is restored. This version is built with MonoGame for Windows and intentionally follows the feel of the older XNA / Windows Phone 7 release.

The original project still matters here—the MonoGame port uses christianhelle/xnapuzzlegame as its behavior reference.

How play works

Start a new game, get a random puzzle image, and move tiles into the empty space until the picture is complete. Once solved, the game shows a congratulations screen with your total time and move count, then lets you play again or return to the main menu.

  • 4×4 sliding puzzle board
  • Eight included puzzle images
  • Live solved-image preview while holding Enter or F1
  • In-game pause/options flow through Esc

Features & Status

Playable now, with desktop polish where it matters

Readable menus

Main menu flow supports keyboard, mouse, mouse wheel, and Tab navigation so the interface stays comfortable across typical desktop input styles.

Responsive layout

The game launches at 1200×720, allows window resizing, and targets modern Windows high-DPI behavior with PerMonitorV2 awareness.

Progress-friendly flow

Session persistence restores progress after exit, and the win screen reports both time and move count for each solved puzzle.

Controls

Keyboard and mouse both feel first-class

Menus

Input Action
↑ / ↓ / Tab / Mouse Wheel Navigate menu entries
Enter / Left Click Select
Esc / Right Click Back or exit

Gameplay

Input Action
Arrow Keys / WASD Slide tiles
Left Click Move an adjacent tile
R / F5 Reshuffle the puzzle
Enter / F1 (hold) Preview the solved image
Esc Open in-game options

Build & Run

Simple Windows setup

Prerequisites

  • .NET 9 SDK or later
  • Windows, targeting net9.0-windows with DirectX

Runtime notes

  • Default window size is 1200×720
  • Window resizing is enabled
  • Session progress is saved and restored between runs

Build

dotnet build ChrisPuzzleGame.slnx

Run

dotnet run --project src\ChrisPuzzleGame.UI\ChrisPuzzleGame.UI.csproj
